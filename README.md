# **Basic RecyclerView App**

## Descripción

Esta es una aplicación Android básica que implementa un `RecyclerView` para mostrar una lista de elementos. Cada ítem tiene una imagen y un nombre, y al hacer clic en cualquiera de ellos, se pueden realizar acciones personalizadas.

<img width="268" height="586" alt="image" src="https://github.com/user-attachments/assets/c30b2fe3-b7e5-4eb5-96c6-6475fbc232f6" />

## Características

- **RecyclerView** con una lista de ítems (imágenes y nombres).
- Adaptador para vincular los datos con las vistas.
- Uso de `ViewBinding` para evitar el uso directo de `findViewById`.
- Implementación de clics en los ítems para realizar una acción.
